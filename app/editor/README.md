# 字幕編輯器使用說明

## 🚀 快速開始

### 訪問編輯器
```
http://localhost:3001/editor
```

## ✨ 功能特色

### 1. 上傳影片
- 點擊「上傳影片」按鈕選擇影片檔案
- 支援格式: MP4, MOV, WebM, AVI
- 建議檔案大小: 500MB 以內

### 2. AI 自動生成字幕
- 在右上角輸入您的 Whisper API Key (選填)
- API Key 會自動儲存在瀏覽器
- 上傳影片後會自動辨識語音並生成中文字幕

### 3. 字幕樣式編輯

#### 調整字體大小
- 使用滑桿調整字體大小 (16px - 48px)
- 即時預覽效果

#### 更改字幕顏色
- 5 種預設顏色可選:
  - ⚪ 白色
  - 🟡 黃色
  - 🔴 紅色
  - 🔵 藍色
  - 🟢 綠色
- 點擊顏色按鈕即可切換

#### 調整字幕位置 ⭐
- 直接在影片上拖曳字幕
- 字幕會顯示「拖曳調整位置」提示
- 拖曳時會有藍色邊框提示

### 4. 下載字幕
- 點擊右上角「下載 SRT」按鈕
- 下載標準 SRT 格式字幕檔案
- 可匯入到其他影片編輯軟體使用

## 📝 使用流程

1. **上傳影片**
   - 點擊「上傳影片」
   - 選擇影片檔案
   - 等待處理完成 (約 30 秒 - 2 分鐘)

2. **查看自動生成的字幕**
   - 右側字幕列表會顯示所有字幕
   - 播放影片時會自動同步顯示

3. **編輯字幕樣式**
   - 調整字體大小
   - 選擇喜歡的顏色
   - 拖曳字幕到理想位置

4. **下載成果**
   - 點擊「下載 SRT」
   - 獲得編輯完成的字幕檔案

## 💡 使用技巧

- **全域樣式**: 所有設定會套用到所有字幕
- **即時預覽**: 所有變更會立即反映在影片上
- **快速測試**: 先上傳一個短影片 (10-30 秒) 測試功能
- **API Key**: 如果沒有 API Key,可以先上傳已有字幕的影片測試編輯功能

## 🔧 進階設定

### API Key 設定
1. 在右上角輸入框輸入您的 Whisper API Key
2. API Key 會自動儲存在瀏覽器
3. 下次訪問時會自動載入

### 字幕位置精確調整
- 拖曳字幕時,位置會以百分比儲存
- 適用於不同解析度的影片
- 可重複調整直到滿意

## ⚠️ 注意事項

- 影片上傳後會暫存在伺服器的 `public/temp` 目錄
- 建議定期清理暫存檔案
- API Key 僅儲存在您的瀏覽器,不會上傳到伺服器
- 拖曳功能目前僅支援桌面裝置

## 🆘 常見問題

### Q: 為什麼上傳影片後沒有字幕?
A: 需要有效的 Whisper API Key 才能自動生成字幕

### Q: 可以編輯字幕文字嗎?
A: MVP 版本暫不支援文字編輯,未來版本會加入

### Q: 字幕位置會儲存嗎?
A: 位置會儲存在字幕資料中,下載 SRT 時會保留樣式資訊

### Q: 支援其他語言嗎?
A: 目前預設為中文,未來版本會支援多語言選擇
