# 进阶编辑器调试日志增强总结

## 更新时间
2025-11-15

## 概述
为进阶编辑器（editor-pro）添加了全面的调试日志系统，确保第二轨道字幕选中问题已完全解决，并提供详细的测试验证和使用文档。

---

## 主要改进

### 1. TimelineDebugLogger 调试工具类
**位置**: `app/editor-pro/page.tsx` (第 17-243 行)

**功能:**
- 格式化的彩色日志输出
- 鼠标事件追踪（MouseDown, MouseMove, MouseUp, Click）
- 拖拽方向判定日志
- 轨道切换日志
- 自动打印测试场景表格
- 可启用/禁用日志

**关键方法:**
```typescript
TimelineDebugLogger.mouseDown()      // 鼠标按下
TimelineDebugLogger.mouseMove()      // 鼠标移动
TimelineDebugLogger.mouseUp()        // 鼠标释放
TimelineDebugLogger.click()          // 点击事件
TimelineDebugLogger.directionDetermined()  // 方向确定
TimelineDebugLogger.trackSwitch()    // 轨道切换
TimelineDebugLogger.printTestScenarios()   // 打印测试表格
TimelineDebugLogger.setEnabled(bool) // 启用/禁用日志
```

### 2. 增强的鼠标事件日志

#### MouseDown 事件
**位置**: `handleTimelineDragStart` 函数

**日志内容:**
- 轨道 ID 和名称
- 字幕 ID 和内容
- 拖拽类型（left/right/move）
- 鼠标位置
- 字幕时间范围

**示例输出:**
```
🖱️ MouseDown | 移动区
  轨道[中文字幕]
  字幕[abc12345:"这是测试字幕..."]
  位置: (450, 280)
  时间: 1.50s - 3.20s
```

---

#### MouseMove 事件
**位置**: `handleMouseMove` 函数

**日志内容:**
- 移动距离（总距离、X 偏移、Y 偏移）
- 拖拽状态（是否超过阈值）
- 拖拽类型
- 拖拽方向

**示例输出:**
```
🔄 MouseMove | 拖拽中
  距离: 25px (X:25px, Y:2px)
  方向: 水平↔️
  类型: move
```

**优化:**
- 只在关键时刻记录，避免日志刷屏
- 首次超过阈值时记录详细信息

---

#### Direction 确定事件
**位置**: `handleMouseMove` 函数

**日志内容:**
- 确定的方向（horizontal/vertical）
- X 和 Y 偏移量

**示例输出:**
```
↔️ Direction | 水平拖拽
  X偏移: 50px
  Y偏移: 3px
```

---

#### TrackSwitch 事件
**位置**: `handleMouseMove` 函数

**日志内容:**
- 源轨道 ID 和名称
- 目标轨道 ID 和名称

**示例输出:**
```
🔀 TrackSwitch
  从: 轨道[中文字幕]
  到: 轨道[英文字幕]
```

---

#### MouseUp 事件
**位置**: `handleMouseUp` 函数

**日志内容:**
- 移动距离
- 判定类型（点击/拖拽）
- 动作类型（选中/调整时间/切换轨道）
- 目标轨道（如适用）

**示例输出:**
```
✅ MouseUp   | 选中字幕
  距离: 2px
  判定: 点击
```

---

#### Click 事件
**位置**: 字幕块 `onClick` 处理

**日志内容:**
- 轨道 ID 和名称
- 字幕 ID 和内容
- 开始时间
- 是否已选中

**示例输出:**
```
🎯 Click     | 新选中
  轨道[中文字幕]
  字幕[abc12345:"测试字幕"]
  时间: 1.50s
```

## 判定标准

### 点击 vs 拖拽
- **移动距离 < 5px** → 点击
- **移动距离 ≥ 5px** → 拖拽

### 水平 vs 垂直拖拽
只在 `dragType === 'move'` 时判断：

- **垂直拖拽**:
  - Y 轴偏移 > 30px
  - 且 Y 轴偏移 > 2 × X 轴偏移

- **水平拖拽**:
  - 其他情况

## 使用方法

### 1. 启动应用
```bash
npm run dev
```

### 2. 访问进阶编辑器
访问 `http://localhost:3000/editor-pro`

### 3. 打开控制台
按 F12 打开浏览器开发者工具，切换到 Console 标签

### 4. 查看日志
执行任意交互操作，查看实时日志输出

### 5. 控制日志
```javascript
// 禁用日志
TimelineDebugLogger.setEnabled(false)

// 启用日志
TimelineDebugLogger.setEnabled(true)

// 重新打印测试场景
TimelineDebugLogger.printTestScenarios()
```

## 技术细节

### 代码修改位置
1. **TimelineDebugLogger 类**: 第 17-243 行
2. **handleTimelineDragStart**: 第 1867-1877 行
3. **handleMouseMove**: 第 1928-1950 行
4. **方向确定**: 第 1974-1979 行
5. **轨道切换**: 第 2020-2026 行
6. **handleMouseUp**: 第 2093-2120 行
7. **onClick**: 第 3501-3509 行

### 性能考虑
- 日志只在开发环境输出（可通过 setEnabled 控制）
- MouseMove 日志经过优化，避免刷屏
- 使用 console.log 而非 alert，不阻塞 UI

### 浏览器兼容性
- 使用标准 Console API，所有现代浏览器支持
- 颜色样式使用 `%c` 语法，广泛支持
- 日志工具不影响核心功能
