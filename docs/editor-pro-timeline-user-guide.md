# 进阶编辑器时间轴交互使用指南

## 概述
本指南详细介绍进阶编辑器（editor-pro）时间轴的所有交互方式，包括选择、拖拽、调整时间、切换轨道等功能。

---

## 快速开始

### 基本操作
1. **选中字幕**: 单击时间轴上的字幕块
2. **移动字幕**: 在字幕块中间区域按住并水平拖拽
3. **调整时长**: 拖拽字幕块的左右边缘
4. **切换轨道**: 在字幕块中间区域按住并垂直拖拽
5. **查看详情**: 双击字幕块打开调整面板

### 键盘快捷键
- `Ctrl + N`: 在当前时间创建新字幕
- `Delete`: 删除选中的字幕
- `Space`: 播放/暂停视频
- `←/→`: 左右移动播放头

---

## 详细交互说明

### 1. 选择字幕

#### 单击选择
**操作:** 单击字幕块中间区域

**效果:**
- 字幕块高亮显示（边框发光）
- 播放头跳转到字幕开始时间
- 右侧属性面板显示字幕详情
- 当前轨道切换为字幕所在轨道

**适用场景:**
- 编辑字幕内容
- 查看字幕属性
- 调整字幕样式

**技巧:**
- 快速点击，不要移动鼠标
- 点击字幕块中央区域最可靠
- 避免点击边缘手柄

---

### 2. 水平移动字幕（调整时间）

#### 整体移动
**操作:** 在字幕块中间区域按住鼠标，水平拖拽

**效果:**
- 字幕块在时间轴上左右移动
- 开始时间和结束时间同步改变
- 字幕持续时长保持不变

**判定条件:**
- 水平移动距离 > 5px
- Y 轴移动 < 30px 或 Y < 2×X

**适用场景:**
- 调整字幕出现时间
- 对齐字幕到语音
- 批量调整时间关系

**技巧:**
- 按住鼠标后先向左右移动，明确水平意图
- 移动时保持水平，避免上下晃动
- 支持精确对齐（30fps 帧对齐）

**注意事项:**
- 字幕不能移动到负时间（< 0s）
- 字幕不能超出视频总时长
- 移动时会自动对齐到帧

---

### 3. 调整字幕边缘（修改时长）

#### 左边缘拖拽
**操作:** 拖拽字幕块左侧的蓝色手柄

**效果:**
- 调整字幕的开始时间
- 结束时间保持不变
- 字幕持续时长改变

**适用场景:**
- 延迟字幕出现时间
- 提前字幕出现时间
- 精确对齐开始时间

**限制:**
- 开始时间不能 >= 结束时间
- 开始时间不能 < 0s

---

#### 右边缘拖拽
**操作:** 拖拽字幕块右侧的蓝色手柄

**效果:**
- 调整字幕的结束时间
- 开始时间保持不变
- 字幕持续时长改变

**适用场景:**
- 延长字幕显示时间
- 缩短字幕显示时间
- 精确对齐结束时间

**限制:**
- 结束时间不能 <= 开始时间
- 结束时间不能 > 视频总时长

---

### 4. 垂直移动字幕（切换轨道）

#### 轨道切换
**操作:** 在字幕块中间区域按住鼠标，垂直拖拽

**效果:**
- 字幕从源轨道移除
- 字幕添加到目标轨道
- 字幕时间保持不变
- 字幕属性保持不变

**判定条件:**
- Y 轴移动 > 30px
- Y 轴移动 > 2 × X 轴移动

**适用场景:**
- 重新组织字幕轨道
- 分离中英文字幕
- 调整字幕层级

**技巧:**
- 按住鼠标后先向上下移动，明确垂直意图
- 移动时保持垂直，避免左右偏移
- 拖拽到目标轨道区域后释放

**注意事项:**
- 必须有至少 2 个轨道才能切换
- 不能切换到同一轨道
- 目标轨道可以有重叠字幕

---

### 5. 双击操作

#### 打开调整面板
**操作:** 双击字幕块中间区域

**效果:**
- 打开时间轴调整对话框
- 可以精确输入开始/结束时间
- 可以查看字幕详细信息

**适用场景:**
- 需要精确时间控制
- 批量调整多个字幕
- 快速编辑时间

---

### 6. 右键菜单

#### 上下文菜单
**操作:** 右键点击字幕块

**功能:**
- 删除字幕
- 复制字幕
- 剪切字幕
- 其他高级操作

**适用场景:**
- 快速删除字幕
- 复制重复内容
- 批量操作

## 多轨道管理

### 创建轨道
1. 点击 "添加轨道" 按钮
2. 输入轨道名称
3. 选择轨道颜色
4. 点击确认

### 删除轨道
1. 右键点击轨道标题
2. 选择 "删除轨道"
3. 确认删除（轨道上的字幕会被移除）

### 重命名轨道
1. 右键点击轨道标题
2. 选择 "重命名"
3. 输入新名称
4. 点击确认

### 调整轨道高度
1. 将鼠标移到轨道底部边缘
2. 光标变为调整大小图标
3. 上下拖拽调整高度

## 时间轴缩放

### 缩放级别
- **Zoom 0.5x**: 查看更长时间范围
- **Zoom 1x**: 默认视图
- **Zoom 2x**: 精确编辑
- **Zoom 4x**: 帧级别编辑

### 调整缩放
1. 使用缩放滑块
2. 或使用快捷键 `+/-`
3. 或使用鼠标滚轮（配合 Ctrl）

### 缩放技巧
- 在需要精确操作时放大
- 在查看整体时缩小
- 缩放后字幕块宽度会改变

## 对齐和吸附

### 帧对齐
所有时间操作都会自动对齐到视频帧（30fps）:
- 手动拖拽时自动对齐
- 输入时间时自动对齐
- 确保时间精确到 1/30 秒

## 判定标准详解

### 点击 vs 拖拽
系统使用移动距离来区分点击和拖拽：

**点击判定:**
- 鼠标按下到释放的移动距离 < 5px
- 触发 onClick 事件
- 选中字幕

**拖拽判定:**
- 鼠标按下到释放的移动距离 ≥ 5px
- 触发拖拽逻辑
- 不触发 onClick

**建议:**
- 点击时快速释放，避免手抖
- 拖拽时先移动再释放
- 使用稳定的输入设备

---

### 水平 vs 垂直拖拽
仅在中间区域拖拽时判断方向（边缘拖拽始终为水平）：

**垂直拖拽判定:**
- Y 轴偏移 > 30px
- 且 Y 轴偏移 > 2 × X 轴偏移
- 触发轨道切换

**水平拖拽判定:**
- 不满足垂直拖拽条件
- 触发时间调整

**示例:**
```
移动 (X:10px, Y:60px) → 垂直拖拽 ✓ (Y > 30 且 60 > 2×10)
移动 (X:40px, Y:40px) → 水平拖拽 ✓ (40 < 2×40)
移动 (X:50px, Y:20px) → 水平拖拽 ✓ (Y < 30)
移动 (X:5px, Y:35px)  → 垂直拖拽 ✓ (Y > 30 且 35 > 2×5)
```

**建议:**
- 垂直拖拽时先向上下移动，明确意图
- 水平拖拽时先向左右移动，明确意图
- 避免斜向拖拽，可能触发非预期行为

## 性能优化

### 大量字幕
处理 100+ 字幕时的建议：

1. **使用虚拟滚动**: 只渲染可见字幕
2. **减少缩放级别**: 降低渲染复杂度
3. **批量操作**: 使用批量编辑而非逐个修改
4. **定期保存**: 避免数据丢失

### 流畅拖拽
确保拖拽流畅的建议：

1. **关闭不必要的浏览器扩展**
2. **使用现代浏览器**（Chrome, Edge, Firefox）
3. **确保硬件加速开启**
4. **避免同时打开多个标签页**

## 故障排查

### 问题 1: 点击字幕无反应

**可能原因:**
- 被判定为拖拽而非点击
- JavaScript 错误
- 字幕被遮挡

**解决方法:**
1. 打开浏览器控制台（F12）
2. 查看是否有错误信息
3. 尝试更快速地点击
4. 检查是否点击了字幕中间区域
5. 刷新页面重试

---

### 问题 2: 拖拽方向错误

**症状:**
- 想水平拖拽，却切换了轨道
- 想垂直拖拽，却调整了时间

**解决方法:**
1. 打开控制台查看日志
2. 检查 "Direction" 日志的 X 和 Y 偏移
3. 尝试更明确的方向移动（完全水平或垂直）
4. 增加移动距离

---

### 问题 3: 字幕块跳动

**症状:**
- 拖拽时字幕块突然跳动到其他位置
- 字幕块位置不跟随鼠标

**可能原因:**
- clickOffsetTime 计算错误
- 缩放级别异常
- 滚动位置错误

**解决方法:**
1. 刷新页面
2. 重置缩放级别为 1x
3. 从字幕块左侧开始拖拽
4. 报告 bug 给开发团队

---

### 问题 4: 轨道切换失败

**症状:**
- 垂直拖拽后字幕没有切换轨道
- 字幕消失或回到原轨道

**可能原因:**
- Y 轴移动不足（< 30px）
- X 轴移动过多（Y < 2×X）
- 只有一个轨道

**解决方法:**
1. 确保至少有 2 个轨道
2. 更垂直地拖拽，减少水平偏移
3. 增加拖拽距离（> 50px）
4. 查看控制台日志确认方向判定

---

### 问题 5: 第二轨道字幕无法选中

**症状:**
- 第一轨道字幕可以选中
- 第二轨道点击无反应

**检查步骤:**
1. 打开控制台
2. 点击第二轨道字幕
3. 查看是否有 MouseDown、MouseUp、Click 日志
4. 检查判定是否为 "点击"

**可能原因:**
- 之前的 bug（已修复）
- 鼠标移动被判定为拖拽

**解决方法:**
1. 确保使用最新版本代码
2. 更快速地点击
3. 如果仍然失败，报告 bug

## 最佳实践

### 1. 工作流程
1. 先导入字幕或创建轨道
2. 播放视频，了解内容
3. 粗调字幕时间（拖拽）
4. 精调字幕时间（边缘拖拽或输入）
5. 组织轨道（垂直拖拽）
6. 最终检查和调整

### 2. 效率技巧
- 使用键盘快捷键加速操作
- 批量操作而非逐个处理
- 利用缩放功能查看不同细节
- 定期保存，避免数据丢失

### 3. 精确控制
- 使用高缩放级别（2x, 4x）
- 双击打开调整面板输入精确时间
- 利用帧对齐确保准确
- 播放验证调整结果

### 4. 组织管理
- 使用有意义的轨道名称
- 不同语言使用不同轨道
- 固定字幕使用独立轨道
- 保持轨道数量合理（< 5）
